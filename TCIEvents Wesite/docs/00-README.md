### The 30-second summary

We're building a **beautiful, mobile-friendly event-discovery website for Turks &
Caicos**, filled with **realistic sample events**, so you can see and feel the
finished experience before we build the money/accounts "engine" behind it.

**Five pages:** Homepage · Discover/Browse · Event Details · Organizer Landing ·
(plus nav, footer & simple placeholder pages).

**Look:** "Tropical Luxury" — boutique-resort feel, ocean colors, gold accents,
big imagery, generous whitespace.

**Next step:** confirm the few open questions in `01-Build-Plan.md` §10, then start
Milestone 0 in `04-Checklist.md`.
